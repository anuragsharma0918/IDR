# IDR - Institute for Digital Risk

A modern, responsive website for the Institute for Digital Risk (IDR), an industry-led training and deployment institute dedicated to preparing individuals for digital and cyber risk roles.

## 📋 Project Overview

IDR bridges the skills gap in the digital and cyber risk landscape by combining industry-leading practices with academic collaboration. This website showcases our mission, training programs, and provides pathways for both prospective students and hiring partners.

## ✨ Features

- **Responsive Design** - Fully responsive layout that works seamlessly on desktop, tablet, and mobile devices
- **Mobile Navigation** - Hamburger menu with smooth transitions for mobile devices
- **Smooth Scrolling** - Elegant scroll behavior with dynamic header effects
- **Contact Form** - Functional contact form with visual feedback
- **Modern UI** - Clean, professional design using the Inter font family
- **Accessibility** - Semantic HTML and proper ARIA labels for screen readers

## 📁 Project Structure

```
.
├── index.html      # Main HTML file with page structure
├── script.js       # JavaScript for interactivity and form handling
├── styles.css      # Complete CSS styling and responsive design
└── README.md       # Project documentation
```

## 🎯 Key Sections

- **Hero Section** - Eye-catching introduction with call-to-action buttons
- **About** - Information about IDR's mission and approach
- **Model** - Details about training and deployment programs
- **Community** - Community engagement information
- **Contact** - Contact form for inquiries

## 💻 Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Modern styling with flexbox and responsive design
- **Vanilla JavaScript** - No frameworks, lightweight interactivity
- **Google Fonts** - Inter font family for typography

## 🚀 Getting Started

1. Clone or download the repository
2. Open `index.html` in your web browser
3. No build process or dependencies required
